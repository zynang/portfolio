MyMovieList

A simple repository for you to store your favourite movies

In this project, I aimed to enhance my React development skills by creating a straightforward webpage that catalogues the movies that I have seen. The page features functionality allowing users to add movies to a 'my favorites' cart. I also implemented features such as an aggregator, sorting and filtering options, and interactive toggle buttons to improve user interaction and site functionality.

  • Date: March 2024
  • Role: Product Designer, Front-End Developer
  • Tools: React JS, HTML/CSS

Background

I love movies and the power they have to transport you to a whole new world. (My favorite movie is the Grand Budapest Hotel, a whimsical comedy mystery!) Out of a personal desire to remember the movies that I've seen and loved, I set out to create a web app that would serve as a movie repository.

Research

I began my development process by first analyzing different websites, which helped me figure out what I liked about different 'favoriting' designs. I looked at Uniqlo, Spotify and Facebook-- and came up with a list of what worked / what did not for each of them.

Design

Based of this review, I decided to implement the following features in my own design:

  4. Convention and Affordances, such as the use of shadow under the "favoriting" button to indicate that it is clickable, and changing the color of a button + pointer to a cursor when the user hovers their mouse over a clikable element.
  5. Explanatory tags that accompany the "favoriting" button, providing users with a clear description of what clicking the button will do
